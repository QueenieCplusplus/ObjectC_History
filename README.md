# ObjectC_History

Object C 是 Mac OS 的一種開發語言，簡稱為 object 或是 Obj-C。誕生於 1980 年代，但到 1996 年方才成為蘋果的程式語言。

在 2004 年，所有的 * Nix 作業系統其 class 和 method 均以 NS 為字守，表示此工具來自 Cocoa 而非其他工具。 (此命名源自於 NeXTStep)
